# Pinterest 大图布局

一个仅作用于 Pinterest 的 Chrome 扩展，可以把瀑布流固定为 2–6 列，让宽屏上的图片显示得更大。

## 本地安装

1. 在 Chrome 地址栏打开 `chrome://extensions`。
2. 打开右上角“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择本项目根目录。
5. 打开或刷新 Pinterest，点击工具栏中的扩展图标选择列数。

## 功能

- 自动、2、3、4、5、6 列布局。
- 单独的启用开关。
- 设置通过 Chrome 同步存储保存。
- 自动响应无限滚动、图片加载、窗口变化和 Pinterest 站内跳转。

## 开发

扩展没有构建步骤和第三方运行时依赖。修改源码后，在 `chrome://extensions` 中点击扩展卡片上的刷新按钮即可。

运行测试：

```bash
npm test
```

完整研发范围见 [研发方案](docs/研发方案.md)。
